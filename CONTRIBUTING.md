# Contributing

Development flow:

1. Create a feature branch
2. Implement changes
3. Run local checks

```bash
make all
```

4. Push branch and create Pull Request
5. Address CodeRabbit review comments
6. Resolve conversations
7. Merge after CI passes

---

## Code Quality Rules

- Code must pass lint/type/test checks
- Tests should be added when applicable
- Avoid committing secrets or internal data
